VPLAssist+ - Task Manager Application
A simple and modern desktop task manager built with C# WinForms.
About
VPLAssist+ is a lightweight task management application that helps you organize and track your daily tasks. It features a clean, modern UI with custom controls and smooth user experience.
Features

Add, Edit, Delete Tasks - Full CRUD operations
Task Details - Store title and description for each task
Mark Complete - Click checkbox to mark tasks done
Search - Find tasks quickly by title
Export/Import - Save and load tasks as JSON files
Task Counter - See total number of tasks at a glance
Modern UI - Clean design with rounded corners and hover effects
Multithreading - Import/export runs in background without freezing UI

Requirements

Windows OS
.NET Framework 4.7.2 or higher
Visual Studio 2019 or later (for development)

How to Use
Adding a Task

Enter task title in the "Title" field
Enter task details in the "Body" field
Click "Add Note" button

Editing a Task

Click on a task in the list
Modify the title or body
Click "Save Changes" button

Marking Complete

Click the checkbox next to any task to mark it complete/incomplete
Completed tasks show strikethrough text and gray color

Searching

Type search term in search box
Click "Search" button
Application will highlight the first matching task

Export Tasks

Click "Export" button
Choose save location
Tasks saved as JSON file with timestamp

Import Tasks

Click "Import" button
Select a JSON file
Choose to Replace or Add to existing tasks
